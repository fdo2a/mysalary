# 2026년 연봉 실수령액표

한국 직장인 기준 세전 연봉별 월 실수령액을 정적 HTML로 생성하는 사이트입니다.

## 포함된 페이지

- `/` 홈: 전체 연봉 실수령액표, 구간별 표, 많이 찾는 연봉 링크
- `/salary/2026`: 2,000만 원부터 10억 원까지 100만 원 단위 전체표
- `/salary/2026/2000-3000` 등: 1,000만 원 구간별 SEO 페이지
- `/salary/2026/3000` 등: 많이 찾는 연봉 상세 페이지
- `/calculator`: 연봉표를 보완하는 조건별 실수령액 확인 페이지
- `/guide/2026`: 계산 기준, 검색 등록, AdSense 안내
- `/privacy`: 개인정보처리방침, 쿠키, 광고, 접속 로그, Google AdSense 사용 가능성 안내
- `/contact`: 운영자 문의 이메일 안내
- `/sitemap.xml`, `/robots.txt`, `/ads.txt`

## 화면 구조

- 데스크톱 표: 세전 연봉, 월 세전, 6개 공제 항목, 총 공제액, 월 실수령액, 연 실수령액
- 모바일 표: 정확한 원 단위 숫자를 한 줄 표에서 바로 확인
- 전체 표: 1,000만 원 단위로 구분하고, 사이에 광고 영역을 삽입
- 상세 페이지: 월급에서 빠지는 항목, 월급 분해 막대, 공제 비중, 연봉 100만 원 증가 대비 월 실수령 증가 그래프
- 공제 금액은 사용자에게 직관적으로 보이도록 `-158,310원`처럼 마이너스 표시
- 전체표 금액은 축약하지 않고 `1,666,667원`, `-79,130원`처럼 정확한 원 단위로 표시
- 연봉 표기: 1억 원 이상은 `11,000만 원` 대신 `1억 1,000만 원`처럼 표시

## 계산 기준

- 2026년 하반기 기준
- 일반 직장인 근로소득
- 연봉을 12개월 균등 지급
- 본인 1명, 비과세 0원 기본값
- 원천징수 100% 기본값
- 산재보험은 근로자 실수령액 차감에서 제외
- 국민연금, 건강보험, 장기요양보험, 고용보험, 근로소득세, 지방소득세 표시

공식 출처는 사이트의 `/guide/2026`과 각 주요 페이지의 계산 기준 섹션에 정리합니다. 국세청, 국민연금공단, 국민건강보험공단, 고용보험 안내를 기준으로 링크를 제공합니다.

실제 급여는 회사 규정, 비과세 항목, 부양가족 수, 원천징수 비율, 사내 공제, 연말정산 결과에 따라 달라질 수 있으므로 사이트 곳곳에 면책 안내를 표시합니다.

근로소득세는 근로소득공제, 기본공제, 4대보험 공제, 근로소득세액공제를 반영한 예상 월 원천징수액입니다. 실제 급여명세서와 맞추려면 국세청 근로소득 간이세액표 Excel을 CSV/JSON으로 변환해 `src/calc.mjs`의 소득세 산출부를 교체하세요.

## 실행

```bash
npm run build
```

생성 결과는 `dist` 폴더에 만들어집니다.

```bash
npm run serve
```

로컬 서버에서 확인합니다.

## 배포 전 수정할 곳

`scripts/build.mjs`의 `site.origin`을 실제 도메인으로 바꾸세요.
문의 이메일도 실제 운영자 이메일로 바꾸세요.

```js
const site = {
  name: "연봉 실수령액표",
  origin: "https://www.mysalary.kr",
  adsenseClient: "ca-pub-9240461016907498",
  contactEmail: "contact@mysalary.kr"
};
```

AdSense 게시자 ID는 `adsenseClient`와 `ads.txt` 생성부에 함께 반영합니다.

## Google 검색 등록

1. 도메인을 연결해 배포합니다.
2. Google Search Console에 도메인을 등록합니다.
3. `/sitemap.xml`을 제출합니다.
4. 홈, `/salary/2026`, 주요 상세 페이지를 URL 검사로 색인 요청합니다.
5. 실제 유입 검색어를 보고 인기 상세 페이지를 늘립니다.
